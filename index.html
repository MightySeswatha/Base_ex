FAQ

How to install?

Unfortunately, I will not be providing a full step-by-step guide. However, installation of the client is simple for the most part. Clich on the link above which applies to you: if you have Java installed on your computer, the first download will give you a file, you can click on it and hit "install". If you do not have Java, download the client from the second link where you must extract the file into a folder, and then click on the "start.bat" file. Feel free to contact a server administrator if you need further assistance with this.

What is the TNTClient for?

The TNTClient was created for TNT games. All features have been created to enhance the gaming experience. If you love TNT games, this client is just for you.

If I use client, can I get a ban?

There's a 99% chance you won't get a ban due to client usage, but you have many other factors for getting a ban. But if for some reason a module will give you a ban, that module will be disabled for everyone.

What does DJFix do and is it safe?

The DJFix module corrects your jump when you jump in games such as: TNT-Run, PVP-Run and Bowspleef. All players who have enabled this module jump to the same height at any ping. The DJFix module cannot be called 'safe' because it changes the speed of your jump. But in theory and in the 2 years since the client came out, not a single ban has been registered due to the use of TNTClient. It's up to you to draw your own conclusions.

How do I open the menu in the client?

The client menu is opened by pressing the right shift key on the keyboard. But this key can be changed to any other key on the keyboard.

What do the suffix and prefix in a name mean?

The ‘[Wi:…]’ prefix denotes the number of win in the game, only visible when player has > 0 wins.
The ‘[Lo:…]’ prefix denotes the number of loses in the game, only visible when the player has > 0 loses.
The ‘[St:…]’ prefix denotes the number of streak in the game, only visible when streak is > 0.
The 'J' suffix indicates that this user is using the DJFix module.
The 'green heart' suffix indicates that this user is also using this client.
